## Summary
CLI (Command Line Interface) adalah sebuah text-based nterface yang cepat dan powerful untuk berkomunikasi dengan komputer secara efektif dan efisien dalam menyelesaikan sebuah task.  

Kenapa harus menggunakan CLI?
1. Granular control dari OS atau aplikasi
2. Faster management
3. Kemampuan untuk melakukan automasi
4. Sangat membantu dalam troubleshooting, seperti network connection

Pada sistem berbasis UNIX, kita dapat menggunakan CLI yang disebut dengan **SHELL**. Salah satu contoh SHELL adalah ```bash``` dan ```zsh```.
Dengan shell, kita dapat membuat sebuah script yang berfungsi untuk melakukan automasi untuk meyelesaikan sebuah task. Ini disebut dengan **Shell Script**.
